# المنظمة الطلابية الجزائرية الحرة - جامعة باتنة 2

## نظرة عامة
موقع رسمي للمنظمة الطلابية الجزائرية الحرة في جامعة باتنة 2. منصة تفاعلية تهدف لتمكين الطلاب وتشجيع المشاركة الطلابية.

## الميزات الرئيسية
- شريط تنقل مع لوغو المنظمة
- عرض شرائح تفاعلي للصور والفيديوهات
- هيكل تنظيمي يعرض الرئيس والمكتب الولائي
- المكاتب الفرعية للكليات والإقامات
- صفحة الأنشطة مع نظام الإعجاب والتقييم والتعليقات
- نموذج تواصل يرسل للبريد الإلكتروني

## البنية التقنية
- React 18 مع RTL support
- TailwindCSS للتصميم
- Lucide Icons
- خط Cairo العربي

## الصفحات
- `index.html` - الصفحة الرئيسية تحتوي على جميع الأقسام

## المكونات
- `Header.js` - الهيدر والقائمة
- `Hero.js` - عرض الشرائح
- `OrgStructure.js` - الهيكل التنظيمي
- `BranchOffices.js` - المكاتب الفرعية
- `Activities.js` - الأنشطة والفعاليات
- `Contact.js` - نموذج التواصل
- `Footer.js` - الفوتر
